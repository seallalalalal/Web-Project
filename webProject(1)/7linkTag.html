連結標籤

超連結:<a> .....</a>

文件的連結方式又因連結目的地之不同而分為「外部連結」和「內部連結」。

(Ⅰ). 外部連結: 連結至另一HTTP檔案或其他主機之home Page

href屬性(Hyperlink Reference):
< a href = "URL" target="window name"> </a>
(URL:uniform resource locator)

Example:
連結至WWW主機(http=hypertexttransfer protocol):<a href="連結網址">說明文字</a>
<a href="https://www.utaipei.edu.tw/">台大的www </a>
結果:
北市大的www

Example:
連結至WWW主機某一文件:<a href="連結網址/文件名稱">說明文字</a>
<a href = "https://www.utaipei.edu.tw/nature.htm1">....</a>
(此文件實際不存在)

Example:
連結至FTP server主機:<a href="FTP Server 位址">說明文字</a>
<a href = "ftp://ftp.tku.edu.tw">淡大ftp server</a>
結果:
淡大ftp server

Example:
寄信至指定之電子郵件位址:<a href="mailto:你的E-Mail">說明文字</a>
<a href = "mailto:lai@www.tmtc.edu.tw">寄信給lai</a>
結果:
寄信給lai

Example:
連結至音樂檔:<a href="音檔案名稱">說明文字</a>
<a href = "北市大校歌.mp3"> 按即可聽音樂</a>
結果:
按即可聽音樂

Example:
連結至圖形檔:<a href="圖檔案名稱">說明文字</a>
<a href = "elephant.gif">按一下即可看圖</a>
結果:
按一下即可看圖

錨(anchor):用以快速的定位到指定元素
<a href="(定位點)">文字、圖形、混合></a>

(Ⅱ). 內部連結: 欲連結目的文件是位於同一份HTML文件中的某個區段

name 屬性: 設定欲連結之之區段名稱目的文件所在處
<a name = "標籤位置"></a>

Example:
指明目的區段所在:<a name = "#top(此處定本頁第一行name為top)"></a>

Example:
連結至本地 8表格標籤.html 檔之#top2處:<a href = "8表格標籤.html#top2">
