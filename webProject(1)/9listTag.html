清單標籤

  • 列示(List)命令

    1. Unordered List: <ul>...</ul>
      無編號列示
      每個項目前加<li>標籤

      (1)style="list-style-type:'type';"屬性:<ul style="list-style-type:'type'>...</ul>
          type有三種，disc、circle、square

      Example:
      <ul list-style-type='disc'>
          <li>
              disc
          </li>
      </ul>
      <ul list-style-type='circle'>
          <li>
              circle
          </li>
      </ul>
      <ul list-style-type='square'>
          <li>
              square
          </li>
      </ul>

      Result:

      • disc
      • circle
      • square

    2. Ordered List:<ol>...</ol>
      有編號列示
      顯示時每個項目都會自動加上前導編號
      每個項目前加<li>標籤

      (1)編號類型:

      <ol type=1> 數字，如1，2，3........
      <ol type=A> 大寫字母，如A，B，C........
      <ol type=a>..... 小寫字母，如a，b，c........
      <ol typeI> 大寫羅馬字母，如I，II，III....
      <ol type=i> 小寫羅馬字母，如i，ii，iii......

      Example:
      <ol>
          <li> 這是第一項</li>
          <li> 這是第二項</li>
          <li> 這是第三項</li>
      </ol>

      Result:

      1. 這是第一項
      2. 這是第二項
      3. 這是第三項

      (2) 標號的起始點
      不一定從第一個開始，可指定起始順序

      Example:
      <ol type=a start=3>
          <li> 這是第一項</li>
          <li> 這是第二項</li>
          <li> 這是第三項</li>
      </ol>

      Result:

      1. 這是第一項
      2. 這是第二項
      3. 這是第三項

    3. DefinitionList:<dl>...</dl>
      是網頁內文的一種排版方式技巧，可製作出縮排的效果
      <dl >...</dl >:定義式列示區段由開始, 結束
      <dt >...</dt >:標記區段中的每個項目(definition term)
      <dd>...</dd>:描述內容，會內縮(definition term)

      Example:
      <dl>
          <dt>標題1</dt>
              <dd> 標題1的敘述一</dd>
              <dd> 標題1的敘述二<dd>
          <dt>標題2</dt>
              <dd> 標題2的敘述一</dd>
              <dd> 標題2的敘述二<dd>
      </dl>

      Result:

      標題1
      標題1的敘述一
      標題1的敘述二
      標題2
      標題2的敘述一
      標題2的敘述二
