路徑

  • 絕對路徑、相對路徑

    類別Example說明
    絕對
    路徑    		C:\Users\User\Desktop\程式語言\HTML\專題\sourceCode\5命令介紹\10路徑.html 本機C槽上的一個HTML檔案
    https://cs.utaipei.edu.tw/網路上一台 WWW Server 上的一個 HTML 檔案
    相對
    路徑    		10路徑.html表同一層目錄下的 10路徑.html 檔案
    ./10路徑.html表同一層目錄下的 10路徑.html 檔案
    5命令介紹/10路徑.html表示 5命令介紹 子目錄下的 10路徑.html
    ../5命令介紹/10路徑.html表示上一層目錄下的 10路徑.html 檔案
    ../sourceCode/5命令介紹/10路徑.html表示上一層目錄 sourceCode 下 5命令介紹 子目錄的 10路徑.html

    通常我們用「.」代表現在目錄，而用「..」代表上一層目錄

    Example:3.html:超連結至1.html，2.html，4.html（分別使用相對路徑、相對路徑、絕對路徑）

    <html>
    <body>
    <a href = ../cs/ut/1.html>[to 1]</a>
    <a href = ../CS/2.html>[to 2]</a>
    <a href = C:\Users\Desktop\程式語言\HTML\home\4.html>[to 4]</a>
    </body>
    </html>
